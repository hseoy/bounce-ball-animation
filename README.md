# Bounce Ball Animation

This is a bounce ball animation using canvas. I used vanilla javascript with babel and webpack.

```
$ git clone https://github.com/hseoy/bounce-ball-animation
$ cd bounce-ball-animation
$ npm install
$ npm start
```
